#  YouTube RAG App — AI-Powered Video Summarizer

This Streamlit web app uses **LangChain**, **OpenAI**, and **YouTube transcripts** to generate concise and insightful summaries from any YouTube video.

---

##  Features

-  Extracts transcripts automatically from YouTube videos  
-  Uses **Retrieval-Augmented Generation (RAG)** for accurate summarization  
-  Powered by **LangChain** and **OpenAI embeddings**  
-  Built with **Streamlit** for an interactive web interface  

---

##  Tech Stack

- **Python**
- **Streamlit**
- **LangChain**
- **OpenAI API**
- **YouTube Transcript API**
